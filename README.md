# Task-4-SQL-for-Data-Analysis
SQL for Data Analysis using the Sample Superstore dataset. Implemented SELECT, WHERE, ORDER BY, GROUP BY, JOINS, Subqueries, Aggregate Functions, Views, and Indexes using DB Browser for SQLite.

## 📖 Project Description

This project focuses on performing SQL-based data analysis using the Sample Superstore dataset. Various SQL operations were implemented to analyze sales performance, customer behavior, and business insights.
